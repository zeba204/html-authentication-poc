# HTML Authentication System POC

This project is a simple HTML-based authentication system.

## Pages:
- login.html
- register.html
- forgot-password.html
- reset-password.html
- dashboard.html

## Features:
- Uses only HTML
- Navigation using anchor tags
- No CSS or JavaScript

## Flow:
Login → Dashboard  
Register → Login  
Forgot Password → Login  
Dashboard → Logout → Login
