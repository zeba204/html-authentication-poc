# Authentication System Styling (Bootstrap UI)

This project is a responsive authentication system built using HTML, Bootstrap 5, and custom CSS.
It includes Login, Registration, Forgot Password, Reset Password, and Dashboard pages.

## Features

* Responsive design (Mobile, Tablet, Desktop)
* Bootstrap 5 UI components
* Login & Registration forms
* Forgot & Reset password pages
* Dashboard with navbar
* Custom CSS styling
* Google Fonts integration
* Gradient background
* Hover effects and transitions
* Icons inside input fields
* Clean and modern UI

## Technologies Used

* HTML5
* Bootstrap 5
* CSS3
* Google Fonts
* Bootstrap Grid System

## Pages

* login.html
* register.html
* forgot-password.html
* reset-password.html
* dashboard.html
